# Full Stack Open - University of Helsinki

This repository contains my solutions to the [Full Stack Open](https://fullstackopen.com/en/) course provided by the University of Helsinki. This course is a deep dive into modern web development using JavaScript.

## 🚀 About the Course
The course focuses on building single-page applications with ReactJS that use REST APIs built with Node.js. It also covers GraphQL, TypeScript, and Docker.

## 📁 Repository Structure
The exercises are organized by parts to keep the progress trackable:

- [x] **Part 0: Fundamentals of Web Apps**
  - Traditional web apps vs. SPAs
  - HTTP GET/POST, AJAX, and CSS
  - Sequence Diagrams (Mermaid syntax)
- [ ] **Part 1: Introduction to React** (Coming Soon)
- [ ] **Part 2: Communicating with Server** (Coming Soon)

## 🛠️ Tech Stack
- **Frontend:** React, Redux, HTML5, CSS3
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Tools:** Git, GitHub, Mermaid.js
